# AI multipurpose bot
## This telegram bot can generate videos and images based on users prompt, convert audio into text and answer your questions in conversational format

# Tools used in this project:
- Python - main programming language
- PyTelegramBotAPI - library for creating telegram bot
- Base64 - library for decrypting photos
- FusionBrain - API for generating photos and videos

# Usage:
- /start - starting the bot
